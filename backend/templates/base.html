How to Use SpotiTransFair
  1. Sign in to YouTube Music
    Open YouTube Music in your browser and ensure you are logged in.
  2. Get Auth Headers

    Open DevTools (F12) â†’ Network tab.

    Filter for browse, perform an action on the page (like clicking "Home"), find a valid POST request to browse, and copy the Request Headers.

    The UI provides a "Validate" button to check your headers.
  3. Provide Spotify Playlist
    Paste the public URL of the Spotify playlist you want to transfer.
  4. Choose Options
    • Market: Affects Spotify search results (defaults to browser language).
    • Privacy: Sets the visibility of the new YouTube Music playlist (PRIVATE by default).
    • Update existing: Optional. Paste a YouTube Music playlist URL/ID to append missing tracks.
  5. Connect & Clone
    Click "Connect" to check the backend status, then "Clone Playlist" to start the process.
  6. Done!
    A dialog will show the results, including any tracks that couldn't be found.
  7. Retry
    For missed tracks, use the review dialog to adjust the query, change filters (Songs/Videos/Uploads), see top suggestions, and add them directly to the playlist.
